### Ship/Ferry Booking
--------------
This is a Ship/Ferry made in C++ with file handling/streaming.
----------------
*ADMIN* 
  - User Accounts 
  - Ships
  - Ships Schedules 
  - Admin Account 
  - User Payment 

*USER* 
  - Book/Buy a ticket
  - Manage his/her ticket 
  - View ship schedules
  - Manage his/her account
